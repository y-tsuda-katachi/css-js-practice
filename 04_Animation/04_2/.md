## アニメーション練習問題２

以下のHTML/CSSをみて、実行結果の通りになるようJavaScriptコードを追加してください。

```HTML
<!doctype html>
<html lang="ja">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Label_1</title>
    <link rel="stylesheet" href="style.css">
    <script src="script.js" defer></script>
  </head>
  <body>
    <h1 id="header-label">読み込み中...</h1>
  </body>
</html>
```

```CSS
body {
    font-family: serif;
    text-align: center;
}
```

<details>
<summary>解答例</summary>

```JS
const headerLabel = document.querySelector("#header-label");

const keyframes = {
    translate: ['0 10px', 0, '0 10px'],
    color: ['#f66', '#fc2', '#0c6', '#0bd', '#f66'],
    rotate: ['y 360deg', 0],
};
const options = {
    duration: 2500,
    easing: 'ease',
    iterations: Infinity,
};

headerLabel.animate(keyframes, options);
```
