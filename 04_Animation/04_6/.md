## アニメーション練習問題６

以下のHTML/CSSをみて、実行結果の通りになるようJavaScriptコードを追加してください。


<details>
<summary>解答例</summary>

```JS
const showImage = (entries) => {
    const keyframes = {
        opacity: [0, 1],
        translate: ['200px 0', 0],
    };
    entries[0].target.animate(keyframes, 600);
};

const imgObserver = new IntersectionObserver(showImage);
imgObserver.observe(document.querySelector("#rand-img"))
```
