## ボタン練習問題４

以下のHTML/CSSをみて、実行結果の通りになるようJavaScriptコードを追加してください。

```HTML
<!doctype html>
<html>
  <head>
    <title>Button_4</title>
    <link rel="stylesheet" href="style.css" />
    <script src="script.js" defer></script>
  </head>

  <body>
    <button id="theme-button">
      テーマを切り替える&nbsp;<span id="icon">☾</span>
    </button>
    <p>JavaScriptを使えば動きのあるWebサイトを作ることができる。</p>
  </body>
</html>
```

```CSS
body {
  transition: 0.5s;
}
#theme-button {
  padding: 1rem 1.5rem;
  font-size: 1rem;
  border: none;
  border-radius: 8px;
  color: #ffffff;
  background: #191970;
  background-size: 200% 100%;
  cursor: pointer;
  transition: background-position 0.2s ease;
}
.dark-theme {
  background: #000000;
  color: #dddddd;
}
```

<details>
<summary>中ヒント💡💡</summary>

以下のメソッドを組み合わせて処理を実装します。
- querySelector() : 要素の取得
- addEventListener() : イベントの設定
- toggle() : （クラスなどの）追加/削除を行う

</details>

<details>
<summary>大ヒント💡💡💡</summary>

以下のような流れで処理を実装します。
```JS
// 1. theme-buttonというIDを持つbutton要素を取得
// 2. iconというIDを持つspan要素を取得
// 3. 1で取得したbutton要素にclickイベントを追加
// 4. body要素からclassListを取得
// 5. body要素のclassListでdark-themeクラスを追加/削除
// 6. 4で取得したclassListにdark-themeクラスが含まれていたら >> a
// 6-a-1. 2で取得したspan要素の文字列を変更
// 6-a-2. 1で取得したbutton要素のbackgroundを変更
// 6. それ以外の場合 >> b
// 6-b-1. 2で取得したspan要素の文字列を変更
// 6-b-2. 1で取得したbutton要素のbackgroundを変更
```

</details>

<details>
<summary>解答例</summary>

```JS
const btn = document.querySelector("#theme-button");
const icon = document.querySelector("#icon");

btn.addEventListener("click", () => {
    const classList = document.body.classList;
    document.body.classList.toggle("dark-theme");

    if (classList.contains("dark-theme")) {
        icon.textContent = "☼";
        btn.style.background = "#ffa500";
    } else {
        icon.textContent = "☾";
        btn.style.background = "#191970";
    }
});
```
