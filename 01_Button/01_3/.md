<details>
<summary>小ヒント💡</summary>

以下のイベントの時に関数を実行します。
- mousemove : カーソルが要素内で動いたときに関数を実行する
- mouseleave : カーソルが要素の範囲から出たときに関数を実行する

</details>

<details>
<summary>中ヒント💡💡</summary>

以下のメソッドを組み合わせて処理を実装します。
- querySelector() : 要素の取得
- addEventListener() : イベントの設定
- getBoundingClientRect() : 要素全体の長方形を取得

</details>

<details>
<summary>大ヒント💡💡💡</summary>

以下のような流れで処理を実装します。
```JS
// 1. more-collection-buttonというIDを持つbutton要素を取得
// 2. 1で取得したbutton要素にmousemoveイベントを追加
// 3. button要素の長方形を取得
// 4. button要素上のカーソル位置(x成分)を計算
// 5. button要素のbackgroundPositionに4で取得した位置を設定
// 6. 1で取得したbutton要素にmouseleaveイベントを追加
// 7. button要素のbackgroundPositionを元に戻す
```

</details>

<details>
<summary>解答例</summary>

```JS
const btn = document.querySelector("#more-collection-button");
btn.addEventListener("mousemove", (e) => {
    const rect = btn.getBoundingClientRect();
    const x = ((e.clientX - rect.left) / rect.width) * 100;
    btn.style.backgroundPosition = `${x}% 0`;
});
btn.addEventListener("mouseleave", () => {
    btn.style.backgroundPosition = "0% 0";
});
```

</details>
